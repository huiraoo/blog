实战

pt-archiver 是 Percona 官方提供的归档工具，用于归档大型表中的记录到另一个表或文件。

功能描述

  • 归档到操作系统文件
  • 归档到同实例不同库下的表（tips：归档操作前，目标表要存在）
  • 归档到不同实例下的表（tips：归档操作前，目标表要存在）

操作示例

pt-online-schema-change

提供在线修改表结构等功能，可以避免在修改表结构的过程中阻塞应用对表数据的访问。

pt-query-digest

提供慢日志分析功能，可以统计出查询执行时间、次数、占比等，并把分析结果输出到文件中，然后借助分析结果找出问题进行优化。
