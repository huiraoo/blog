MySQL 基础

基本概念

数据库：物理操作系统或其他形式文件类型的集合。在mysql下数据库文件可以是frm，myd，myi，ibd结尾的文件。

数据库实例：由数据库后台进程/线程以及一个共享内存区组成。数据库实例才是真正用来操作数据库文件的。
mysql数据库：是单进程多线程的程序，与sql server比较类似。也就是说，Mysql数据库实例在系统上的表现就是一个进程

数据库应用主要有两类：OLAP（联机分析处理）和OLTP（联机事务处理）

OLAP的主要特点是:

  • 实时性要求不高
  • 数据量大
  • 并发量小

OLTP的主要特点是:

  • 实时性要求高
  • 数据量小
  • 高并发
  • 要求满足ACID
