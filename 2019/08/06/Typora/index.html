简介

Markdown 是一种轻量级标记语言，创始人是约翰·格鲁伯（John Gruber），它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的 HTML 文档。Typora 对 Markdown 的完美支持，丰富的主题，高效的快捷键操作，无一不令人爱不释手。

常用快捷键

  • 加粗： Ctrl/Cmd + B
  • 标题： Ctrl/Cmd + H
  • 插入链接： Ctrl/Cmd + K
  • 插入代码： Ctrl/Cmd + Shift + C
  • 行内代码： Ctrl/Cmd + Shift + K
  • 插入图片： Ctrl/Cmd + Shift + I
  • 无序列表： Ctrl/Cmd + Shift + L
  • 撤销： Ctrl/Cmd + Z
  • 一级标题：快捷键为Crtl + 1，以此类推
