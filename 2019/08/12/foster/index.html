第一周（2019/08/12）

本周学习内容：
1、数据库基础概念及 MySQL 简介。
2、Linux 常用命令。
3、Linux 下安装 MySQL 5.7 版本数据库。
4、操作 MySQL 常用工具，学会对新增用户进行授权。
5、搭建个人技术博客，每日工作日报通过博客文章形式发送。

本周输出：
1、学习总结：https://mino-yn.github.io/blog/archives/
2、MySQL 运维工单2个。

第二周（2019/08/19）

本周学习内容：
1、MySQL 数据库备份工具 XtraBackup 的学习实践。
2、增、删、改、查常用 SQL 语句操作及语法。
3、MySQL 常用内置函数的使用：COUNT()、MAX()、MIN()、SUM() 等。

本周输出：
1、学习总结：https://mino-yn.github.io/blog/archives/
2、MySQL 运维工单2个。

第三周（2019/08/26）

本周学习内容：
1、MySQL 数据库备份工具 mysqldump 的学习与实践。
2、配置 SSH 互信的原理与实践
3、MySQL 高级查询：联结、组合查询、子查询等
4、MySQL 存储过程、游标、触发器等的使用

本周工作输出：
1、学习总结：https://mino-yn.github.io/blog/archives/
