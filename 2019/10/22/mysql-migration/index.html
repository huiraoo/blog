MySQL · 使用 XtraBackup 迁移数据库

发表于 2019-10-22 | 阅读次数：

背景

需求：xx 项目，将生产环境数据库迁移至测试环境，迁移数据量约 185GB

环境：

  • 生产库：10.138.22.218:3100，MySQL 5.6.27 版本
  • 测试库：10.138.22.192:3306，MySQL 5.6.27 版本

Tips：数据库版本一致，且配置文件 my.cnf 一致，才能使用 XtraBackup

实践

准备条件：

  • 生产库和测试库服务器都已安装 XtraBackup
  • 已配置互信

1. 备份主库（约耗时 30min）

2. 传输备份数据到测试服务器 （约耗时 30min）

3. 恢复全备数据（约耗时 60min）
