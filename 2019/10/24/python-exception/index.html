异常处理

  • try..except：一般来说我们会将所有可能引发异常或错误的语句放在 try 代码块中，将错误处理代码放置在 except 代码块中。
  • finally：总是会执行代码块，不论程序正常还是异常都会执行到。
  • raise：抛出异常，你能够引发的错误或异常必须是直接或间接从属于 Exception（异常） 类的派生类。
  • as：异常类别名。

示例：exceptions_finally.py
