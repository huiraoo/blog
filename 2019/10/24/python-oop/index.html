Python · 面向对象编程

发表于 2019-10-24 | 阅读次数：

高级特性：切片、迭代、列表生成式、生成器、迭代器

面向对象编程：数据封装、继承、多态

类和实例：创建实例的模板，而实例则是一个一个具体的对象，各个实例拥有的数据都互相独立，互不影响。

方法：就是与实例绑定的函数，和普通函数不同，方法可以直接访问实例的数据。

继承：最大好处是子类获得了父类的全部功能，这样就不必重零做起，子类只需要新增自己特有的方法，也可以把父类不适合的方法覆盖重写。
